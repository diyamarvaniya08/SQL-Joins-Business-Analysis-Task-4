# SQL-Joins-Business-Analysis-Task-4
This project demonstrates SQL JOIN operations using a retail dataset.

# Concepts Covered 
- INNER JOIN
- LEFT JOIN
- Foreign keys
- Revenue analysis
- Business-driven SQL queries
  
# Queries Performed
- Revenue per product
- Revenue category-wise
- sales in east region between dates
- Top customers by profit
- Highest profitable product by category
- Customer with no orders
- Common Records between relations

# Deliverables
- joins_queries.sql
- joined_output.csv
- insights.txt

# Outcome
Learned how to combine multiple tables and answer real business questions using SQL.
